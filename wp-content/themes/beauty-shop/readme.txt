== Description ==

Beauty Shop is a fast, clean, responsive WooCommerce enabled WPML and RTL ready theme built for ecommerce stores, any online businesses, accessories, clothing, Dokan, fashion, interior, marketplace, minimal, modern, multi-purpose, responsive, shoes, vendor, shops and stores. Using Beauty shop theme you can use popular page builders like Elementor, KingComposer, Beaver Builder, Thrive Architect, Divi, Brizy, Visual Composer, etc. to build your online presence. Beauty Shop is compatible with product compare, product wishlist, WooCommerce Variation Swatches and other popular plugins. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
